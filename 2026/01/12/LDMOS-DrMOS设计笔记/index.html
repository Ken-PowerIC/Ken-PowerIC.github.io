LDMOS与DrMOS设计初探

发表于 2026-01-12 分类于 学习笔记

这是我的第一篇技术博客。测试一下公式和代码的渲染效果。

1. 导通电阻公式 (LaTeX测试)

在 LDMOS 设计中，我们最关注 $R_{on,sp}$。线性区的电流公式如下：

$$I_{ds} = \mu_n C_{ox} \frac{W}{L} (V_{gs} - V_{th}) V_{ds}$$

如果是为了优化 $R_{on}$，我们需要权衡漂移区长度 $L_d$。

2. 仿真模型 (代码高亮测试)

以下是一个简单的 Verilog-A 示例，用于模拟过温保护（OTP）：

module otp_sensor(out, temp);
    output out;
    input temp;
    electrical out, temp;
    
    parameter real threshold = 145.0; // 关断阈值 145度
    
    analog begin
        if (V(temp) > threshold)
            V(out) <+ 1.0;
        else
            V(out) <+ 0.0;
    end
endmodule
